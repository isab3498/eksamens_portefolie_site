Tema: Grundlæggende animation

I emnet grundlæggende animation lavede vi et spil i grupper for foreningen sex og samfund - privatSnak, der skulle sætte fokus på billededeling og andre vigtige spørgsmål omkring lovgivning for seksuelle områder, som målgruppen 13-15 år kunne have. Projektforløbet blev struktureret med trello board og scrum meetings, hvor vi dagligt evaluerede hvad status var på projektet. Via et aktivitetsdiagram og State Machine Diagram, samt et layoutdiagram, kortlage vi spilles struktur og opbygning.

I designprocessen fandt vi inspiration i stilen fra tegnefilmen Rick and Morty og det spraglede farvevalg de bruger.

Spillet blev kodet i html, css og javascript. Vi afsluttede projektet efter test af vores produktet på målgruppen. I testen stillede vi vores testpersoner 2 skalaspørgsmål hvor de skulle svare på hvor sjovt det var, og hvor meget de lærte af det.
